## Guide Penggunaan HiveDB di Flutter
1. Tambahkan dependensi HiveDB ke pubspec.yaml
```yaml
dependencies:
  hive: ^2.2.3
  hive_flutter: ^1.1.0

dev_dependencies:
  hive_generator: ^1.1.3
  build_runner: ^2.1.11
```
2. Ubah fungsi main menjadi asynchronus
